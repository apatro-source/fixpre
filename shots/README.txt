FIXPRE — TANITIM SAYFASI GÖRSELLERİ (vitrin)
============================================

Bu klasördeki .svg dosyaları, tanıtım sayfasındaki "Basit arayüz, çok çözüm"
bölümünde otomatik kullanılır. Hepsi HAZIR: uygulamanın arayüzünü İngilizce
etiketlerle temsil eden, marka renkli (mor-pembe) temiz mockup'lardır.
Yani şu an ekstra bir şey yapmana gerek yok — sadece push et.

MEVCUT DOSYALAR
---------------
son-saat.svg    -> Görev + "Due 18:00" + "1 hour to deadline" bildirimi
tekrar.svg      -> Tekrarlayan görev: Daily/Weekly/Monthly + gün seçimi
bildirim.svg    -> Telefon bildirimleri (New task / Announcement)
izin.svg        -> Leave request + Approve/Reject + overtime bakiyesi
vardiya.svg     -> Haftalık vardiya tablosu (A/B/Off) + swap notu
hiyerarsi.svg   -> Manager -> Chef -> Staff hiyerarşisi

----------------------------------------------------------
1) İlgili ekranın görüntüsünü al (kişisel veri görünen yerleri bulanıklaştır).
2) Aynı adla ama .png/.jpg olarak kaydetmek istersen bana söyle; app.js'teki
   uzantıyı güncelleyeyim. Ya da SVG'yi silip aynı isimle .svg koyabilirsin.
